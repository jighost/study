			● 템플릿 적용시키기
				
				1.템플릿 파일을 blog -> static -> blog 파일에 붙여넣는다.

				2.html 파일만 blog -> templates -> blog 파일에 붙여넣는다.

			● base 파일 적용
				1. {# {% load static %} #} 으로 static을 불러온다.

				2.기본이 될 html을 복붙한다. (ex index.html

				3.script 와 link 의 경로를 현재 파일로 바꿔준다.
					.. : 이전 파일로 이동

			● 새로운 파일 만들기
				1.blog -> templates -> blog에 새 html 파일을 만든다(이름은 자유)

				2.url.py에 새로운 path를 추가한다 
					path('이름', views.이름, name='이름'),

				3.views.py에 새로운 def를 추가한다
					def 이름(request):
		    			return render(request, 'blog/이름.html', {})

		    		4.base.html에 만든 파일을 적용시킨다.

		    		*{# {% extends 'blog/base.html' %} #} base파일을 가져오기 위해 쓰는 명령어이다.
		    		 맨 윗줄에 넣는다.

		    ● block 
		    	 block을 지정하여 공간을 할당하고 다른 페이지에 삽입시킨다.
		    		(ex base.html에 block을 만들고 다른페이지를 만들어 block을 불러오면 동시에 보인다.
		
		{# {% block 이름 %} #}
		{# {% endblock %} #} : block을 끝냄
{% endblock %}
